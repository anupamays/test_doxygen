◆ _parse_stat_file()

psutil._pslinux.Process._parse_stat_file ( self)
protected

Parse /proc/{pid}/stat file and return a dict with various process info.

Using "man proc" as a reference: where "man proc" refers to position N always subtract 3 (e.g ppid position 4 in 'man proc' == position 1 in here). The return value is cached in case oneshot() ctx manager is in use.

◆ memory_maps()

psutil._pslinux.Process.memory_maps ( self)

Return process's mapped memory regions as a list of named tuples.

Fields are explained in 'man proc'; here is an updated (Apr 2012) version: http://goo.gl/fmebo

/proc/{PID}/smaps does not exist on kernels < 2.6.14 or if CONFIG_MMU kernel configuration option is not enabled.
