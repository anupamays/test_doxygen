Detailed Description

CredSSP TSCspDataDetail structure.

The TSCspDataDetail structure contains CSP information used during smart card logon.

The ASN.1 definition for the TSCspDataDetail structure is defined in MS-CSSP 2.2.1.2.2.1 TSCspDataDetail_:: 

TSCspDataDetail ::= SEQUENCE {
        keySpec       [0] INTEGER,
        cardName      [1] OCTET STRING OPTIONAL,
        readerName    [2] OCTET STRING OPTIONAL,
        containerName [3] OCTET STRING OPTIONAL,
        cspName       [4] OCTET STRING OPTIONAL
}
Parameters
key_specThe specification of the user's smart card.
card_nameThe name of the smart card.
reader_nameThe name of the smart card reader.
container_nameThe name of the certificate container.
csp_nameThe name of the CSP.
