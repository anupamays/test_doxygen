Detailed Description

The current environment is externally managed.

This is raised when the current environment is externally managed, as defined by PEP 668_. The EXTERNALLY-MANAGED configuration is checked and displayed when the error is bubbled up to the user.

Parameters
errorThe error message read from EXTERNALLY-MANAGED.
