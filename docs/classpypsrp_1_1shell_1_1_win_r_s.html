A WinRS shell instance.

This is used by Process to spawn a new command/ process on the raw WinRS shell.

Parameters
wsmanThe pypsrp.wsman.WSMan instance to send commands over
resource_uriThe resource URI of the shell, defaults to the WinRS cmd shell
idThe ID of the shell, if not specified a dynamic ID will be generated by the host
input_streamsThe input streams available to the shell
output_streamsThe output streams available to the shell
codepageThe codepage of the shell
environmentA dictionary that contains environment key/values that are created for the shell instance
idle_time_outThe idle timeout in seconds of the shell
lifetimeThe total lifetime of the shell
nameThe name (description) of the shell
no_profileWhether to create the shell with the user profile active or not
working_directoryThe default working directory of the created
shell
