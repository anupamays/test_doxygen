Get the hash of some data, using a particular hash algorithm, if specified.

Parameters
dataThe data to be hashed.
type of data: bytes
hasherThe name of a hash implementation, supported by hashlib, or None. Examples of valid values are 'sha1', 'sha224', 'sha384', 'sha256', 'md5' and 'sha512'. If no hasher is specified, the hasher attribute of the :class:InstalledDistribution instance is used. If the hasher is determined to be None, MD5 is used as the hashing algorithm.
Returns
s The hash of the data. If a hasher was explicitly specified, the returned hash will be prefixed with the specified hasher followed by '='.
return type of : str
