[MS-PSRP] 2.2.3.25 PSCredential https://msdn.microsoft.com/en-us/library/ee442231.aspx

Represents a username and a password. As the password is a secure string, the RunspacePool must have already exchanged keys with .exchange_keys() method.

Parameters
usernameThe username (including the domain if required)
passwordThe password for the user, this should be a unicode string in order to make sure the encoding is correct
