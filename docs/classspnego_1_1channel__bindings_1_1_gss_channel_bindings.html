Detailed Description

Python representation of a GSSAPI Channel Binding data structure.

A common representation for a GSSAPI Channel Binding data structure that can be passed into a context to bind against that security context. Channel bindings are tags that identify the particular data channel that is used. Because these tags are specific to the originator and recipient applications, they offer more proof of a valid identity. Most HTTPS based authentications just set the application data to b'tls-server-end-point:<cert hash>'.
