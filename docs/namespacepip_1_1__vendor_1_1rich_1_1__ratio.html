◆ ratio_resolve()

List[int] pip._vendor.rich._ratio.ratio_resolve ( int total,
Sequence[Edge] edges )

Divide total space to satisfy size, ratio, and minimum_size, constraints.

The returned list of integers should add up to total in most cases, unless it is impossible to satisfy all the constraints. For instance, if there are two edges with a minimum size of 20 each and total is 30 then the returned list will be greater than total. In practice, this would mean that a Layout object would clip the rows that would overflow the screen height.
