Detailed Description

Report violations of a given JSON schema.

This class extends :exc:~fastjsonschema.JsonSchemaValueException by adding the following properties:

  • summary: an improved version of the JsonSchemaValueException error message with only the necessary information)
  • details: more contextual information about the error like the failing schema itself and the value that violates the schema.

Depending on the level of the verbosity of the logging configuration the exception message will be only summary (default) or a combination of summary and details (when the logging level is set to :obj:logging.DEBUG).
