◆ memoize()

psutil._common.memoize ( fun)

A simple memoize decorator for functions supporting (hashable) positional arguments.

It also provides a cache_clear() function for clearing the cache:

>>> @memoize
... def foo()
... return 1
...
>>> foo()
1
>>> foo.cache_clear()
>>>

It supports:

- functions
  • classes (acts as a @singleton)
    - staticmethods
    - classmethods
    It does NOT support:
    - methods

◆ memoize_when_activated()

psutil._common.memoize_when_activated ( fun)

A memoize decorator which is disabled by default.

It can be activated and deactivated on request. For efficiency reasons it can be used only against class methods accepting no arguments.

>>> class Foo:
... @memoize
... def foo()
... print(1)
...
>>> f = Foo()
>>> # deactivated (default)
>>> foo()
1
>>> foo()
1
>>>
>>> # activated
>>> foo.cache_activate(self)
>>> foo()
1
>>> foo()
>>> foo()
>>>
