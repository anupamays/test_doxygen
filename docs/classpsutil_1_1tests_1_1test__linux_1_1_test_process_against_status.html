Detailed Description

/proc/pid/stat and /proc/pid/status have many values in common.

Whenever possible, psutil uses /proc/pid/stat (it's faster). For all those cases we check that the value found in /proc/pid/stat (by psutil) matches the one found in /proc/pid/status.
