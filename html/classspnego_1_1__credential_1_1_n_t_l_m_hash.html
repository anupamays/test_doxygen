Detailed Description

NTLM LM/NT Hash credential.

Used with :class:NTLMProxy for NTLM authentication backed by an NT and/or LM hash value. In modern iterations of NTLM only the NT hash needs to be specified and LM is ignored but both can be specified and the NTLM code will use them as necessary.
