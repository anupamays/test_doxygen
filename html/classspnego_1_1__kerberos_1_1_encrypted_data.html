Detailed Description

Kerberos EncryptedData container.

The ASN.1 definition for the EncryptedData structure is defined in RFC 4120 5.2.9_:: 

EncryptedData   ::= SEQUENCE {
    etype   [0] Int32 -- EncryptionType --,
    kvno    [1] UInt32 OPTIONAL,
    cipher  [2] OCTET STRING -- ciphertext
}
Parameters
etypeThe encryption algorithm that was used to encipher the cipher.
kvnoThe version number of the key under which data is encrypted. It is only present in messages encrypted under long lasting keys.
cipherThe enciphered text.

.. RFC 4120 5.2.9: https://www.rfc-editor.org/rfc/rfc4120#section-5.2.9
