setuptools._distutils.msvc9compiler.find_vcvarsall ( version)

Find the vcvarsall.bat file.

At first it tries to find the productdir of VS 2008 in the registry. If that fails it falls back to the VS90COMNTOOLS env var.

◆ get_build_version()

setuptools._distutils.msvc9compiler.get_build_version ( )

Return the version of MSVC that was used to build Python.

For Python 2.3 and up, the version number is included in sys.version. For earlier versions, assume the compiler is MSVC 6.
