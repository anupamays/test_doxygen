cffi.verifier.Verifier.load_library ( self)

Get a C module from this Verifier instance.

Returns an instance of a FFILibrary class that behaves like the objects returned by ffi.dlopen(), but that delegates all operations to the C module. If necessary, the C code is written and compiled first.
