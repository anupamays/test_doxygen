Detailed Description

A collection of AV_PAIR structures for the TargetInfo field.

The AV_PAIR_ structure defines an attribute/value pair and sequences of these pairs are using in the :class:Challenge and :class:Authenticate messages. The value for each pair depends on the AvId specified. Each value can be get/set/del like a normal dictionary where the key is the AvId of the value.
