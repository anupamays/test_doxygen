◆ is_installable_dir()

bool pip._internal.utils.misc.is_installable_dir ( str path)

Is path is a directory containing pyproject.toml or setup.py?

If pyproject.toml exists, this is a PEP 517 project. Otherwise we look for a legacy setuptools layout by identifying setup.py. We don't check for the setup.cfg because using it without setup.py is only available for PEP 517 projects, which are already covered by the pyproject.toml check.

str pip._internal.utils.misc.redact_netloc ( str netloc)

Replace the sensitive data in a netloc with "****", if it exists.

For example:

  • "user:pass@example.com" returns "user:****@example.com"
  • "accesstoken@example.com" returns "****@example.com"

◆ strtobool()

int pip._internal.utils.misc.strtobool ( str val)

Convert a string representation of truth to true (1) or false (0).

True values are 'y', 'yes', 't', 'true', 'on', and '1'; false values are 'n', 'no', 'f', 'false', 'off', and '0'. Raises ValueError if 'val' is anything else.
