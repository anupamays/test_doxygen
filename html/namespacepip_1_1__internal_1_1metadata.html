bool pip._internal.metadata._should_use_importlib_metadata ( )
protected

Whether to use the importlib.metadata or pkg_resources backend.

By default, pip uses importlib.metadata on Python 3.11+, and pkg_resourcess otherwise. This can be overridden by a couple of ways:

  • If environment variable _PIP_USE_IMPORTLIB_METADATA is set, it dictates whether importlib.metadata is used, regardless of Python version.
  • On Python 3.11+, Python distributors can patch importlib.metadata to add a global constant _PIP_USE_IMPORTLIB_METADATA = False. This makes pip use pkg_resources (unless the user set the aforementioned environment variable to True).
