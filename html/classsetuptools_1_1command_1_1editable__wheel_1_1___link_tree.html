Detailed Description

Creates a .pth file that points to a link tree in the auxiliary_dir.

This strategy will only link files (not dirs), so it can be implemented in any OS, even if that means using hardlinks instead of symlinks.

By collocating auxiliary_dir and the original source code, limitations with hardlinks should be avoided.
