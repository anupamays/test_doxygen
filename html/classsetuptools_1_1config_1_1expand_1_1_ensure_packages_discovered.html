Detailed Description

Some expand functions require all the packages to already be discovered before they run, e.g.

:func:read_attr, :func:resolve_class, :func:cmdclass.

Therefore in some cases we will need to run autodiscovery during the evaluation of the configuration. However, it is better to postpone calling package discovery as much as possible, because some parameters can influence it (e.g. package_dir), and those might not have been processed yet.
