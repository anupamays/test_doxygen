Detailed Description

A representation of a supported elliptic curve.

@cvar _curves: :py:obj:None until an attempt is made to load the curves. Thereafter, a :py:type:set containing :py:type:_EllipticCurve instances each of which represents one curve supported by the system. @type _curves: :py:type:NoneType or :py:type:set
