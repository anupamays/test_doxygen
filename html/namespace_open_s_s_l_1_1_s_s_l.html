◆ _make_requires()

OpenSSL.SSL._make_requires ( flag,
error )
protected

Builds a decorator that ensures that functions that rely on OpenSSL functions that are not present in this build raise NotImplementedError, rather than AttributeError coming out of cryptography.

Parameters
flagA cryptography flag that guards the functions, e.g. Cryptography_HAS_NEXTPROTONEG.
errorThe string to be used in the exception if the flag is false.
