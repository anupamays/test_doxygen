Get the "scheme" corresponding to the input parameters.

The distutils documentation provides the context for the available schemes: https://docs.python.org/3/install/index.html#alternate-installation

Parameters
dist_namethe name of the package to retrieve the scheme for, used in the headers scheme path
userindicates to use the "user" scheme
homeindicates to use the "home" scheme and provides the base directory for the same
rootroot under which other directories are re-based
isolatedequivalent to –no-user-cfg, i.e. do not consider ~/.pydistutils.cfg (posix) or ~/pydistutils.cfg (non-posix) for scheme paths
prefixindicates to use the "prefix" scheme and provides the base directory for the same
