Class that handles WSMan transport over HTTP.

This exposes a method per action that takes in a resource and the header metadata required by that resource.

This is required by the pypsrp.shell.WinRS and pypsrp.powershell.RunspacePool in order to connect to the remote host. It uses HTTP(S) to send data to the remote host.

https://msdn.microsoft.com/en-us/library/cc251598.aspx
