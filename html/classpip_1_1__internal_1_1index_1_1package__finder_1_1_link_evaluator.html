Tuple[LinkType, str] pip._internal.index.package_finder.LinkEvaluator.evaluate_link ( self,
Link link )

Determine whether a link is a candidate for installation.

Returns
A tuple (result, detail), where result is an enum representing whether the evaluation found a candidate, or the reason why one is not found. If a candidate is found, detail will be the candidate's version string; if one is not found, it contains the reason the link fails to qualify.
