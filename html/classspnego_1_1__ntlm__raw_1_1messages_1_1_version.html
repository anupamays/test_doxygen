Detailed Description

A structure contains the OS information.

The VERSION_ structure contains operating system version information that SHOULD be ignored. This structure is used for debugging purposes only and its value does not affect NTLM message processing. It is populated in the NTLM messages only if NTLMSSP_NEGOTIATE_VERSION (NegotiateFlags.version) is negotiated.

Parameters

minorThe 8-bit unsigned int for the version minor part.
buildThe 16-bit unsigned int for the version build part.
revisionAn 8-bit unsigned integer for the current NTLMSSP revision. This field SHOULD be 0x0F.
