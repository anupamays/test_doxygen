Detailed Description

Windows FILETIME structure.

FILETIME structure representing number of 100-nanosecond intervals that have elapsed since January 1, 1601 UTC. This subclasses the datetime object to provide a similar interface but with the nanosecond attribute.

Attrs
nanosecond The number of nanoseconds (< 1000) in the FileTime. Note this only has a precision of up to 100 nanoseconds.
