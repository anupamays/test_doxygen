Detailed Description

disk_partitions() may return partitions with device == "/dev/root" or "rootfs".

This container class uses different strategies to try to obtain the real device path. Resources: https://bootlin.com/blog/find-root-device/ https://www.systutorials.com/how-to-find-the-disk-where-root-is-on-in-bash-on-linux/
