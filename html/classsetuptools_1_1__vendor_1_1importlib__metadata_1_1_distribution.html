setuptools._vendor.importlib_metadata.Distribution._convert_egg_info_reqs_to_simple_reqs ( sections)
staticprotected

Historically, setuptools would solicit and store 'extra' requirements, including those with environment markers, in separate sections.

More modern tools expect each dependency to be defined separately, with any relevant extras and environment markers attached directly to that requirement. This method converts the former to the latter. See _test_deps_from_requires_text for an example.
