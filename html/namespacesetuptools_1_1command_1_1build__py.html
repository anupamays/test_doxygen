◆ build_package_data()

setuptools.command.build_py.build_package_data ( self)

build_meta may try to create egg_info outside of the project directory, and this can be problematic for certain plugins (reported in issue #3500).

Extensions might also include between their sources files created on the build_lib and build_temp directories.

This function should filter this case of invalid files out.
