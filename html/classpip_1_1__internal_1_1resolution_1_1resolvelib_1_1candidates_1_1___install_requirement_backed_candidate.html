Detailed Description

A candidate backed by an InstallRequirement.

This represents a package request with the target not being already in the environment, and needs to be fetched and installed. The backing InstallRequirement is responsible for most of the leg work; this class exposes appropriate information to the resolver.

Parameters
linkThe link passed to the InstallRequirement. The backing InstallRequirement will use this link to fetch the distribution.
source_linkThe link this candidate "originates" from. This is different from link when the link is found in the wheel cache. link would point to the wheel cache, while this points to the found remote link (e.g. from pypi.org).
