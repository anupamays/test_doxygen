bool setuptools._path.same_path ( _Path p1,
_Path p2 )

Differs from os.path.samefile because it does not require paths to exist.

Purely string based (no comparison between i-nodes).

>>> same_path("a/b", "./a/b")
True
>>> same_path("a/b", "a/./b")
True
>>> same_path("a/b", "././a/b")
True
>>> same_path("a/b", "./a/b/c/..")
True
>>> same_path("a/b", "../a/b/c")
False
>>> same_path("a", "a/b")
False
