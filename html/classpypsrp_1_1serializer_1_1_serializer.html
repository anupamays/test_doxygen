typing.Any value,
typing.Optional[ObjectMeta] metadata = None,
typing.Optional[ET.Element] parent = None,
bool clear = True )

Serializes a raw value or class into an XML Element that can be sent over to the remote host.

Parameters
valueThe value to serialize
metadataAny extra metadata to control how to serialize the value, if None then the value will be inferred by the type
parentWhether to append the element onto a parent element
clearWhether to clear the Obj and TN reference map, this should only be True when initially calling serialize
Returns
The XML Element from the serializied value
