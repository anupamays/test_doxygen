None OpenSSL.rand.add ( bytes buffer,
int entropy )

Mix bytes from string into the PRNG state.

The entropy argument is (the lower bound of) an estimate of how much randomness is contained in string, measured in bytes.

For more information, see e.g. :rfc:1750.

This function is only relevant if you are forking Python processes and need to reseed the CSPRNG after fork.

Parameters
bufferBuffer with random data.
entropyThe entropy (in bytes) measurement of the buffer.
