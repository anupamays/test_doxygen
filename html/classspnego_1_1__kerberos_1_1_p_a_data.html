Detailed Description

Kerberos PA-DATA.

The ASN.1 definition for the PA-DATA structure is defined in RFC 4120 5.2.7_:: 

PA-DATA         ::= SEQUENCE {
    -- NOTE: first tag is [1], not [0]
    padata-type     [1] Int32,
    padata-value    [2] OCTET STRING -- might be encoded AP-REQ
}
Parameters
data_typeIndicates the type of data the value represents.
valueThe PAData value, usually the DER encoding of another message.
   data_type (Union[int, KerberosPADataType]): See args.

.. RFC 4120 5.2.7: https://www.rfc-editor.org/rfc/rfc4120#section-5.2.7
