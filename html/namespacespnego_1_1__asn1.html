bytes spnego._asn1.pack_asn1 ( TagClass tag_class,
bool constructed,
typing.Union[TypeTagNumber, int] tag_number,
bytes b_data )

Pack the ASN.1 value into the ASN.1 bytes.

Will pack the raw bytes into an ASN.1 Type Length Value (TLV) value. A TLV is in the form:

| Identifier Octet(s) | Length Octet(s) | Data Octet(s) |

Parameters
tag_classThe tag class of the data.
constructedWhether the data is constructed (True), i.e. contains 0, 1, or more element encodings, or is primitive (False).
tag_numberThe type tag number if tag_class is universal else the explicit tag number of the TLV.
b_dataThe encoded value to pack into the ASN.1 TLV.
Returns
bytes The ASN.1 value as raw bytes.
