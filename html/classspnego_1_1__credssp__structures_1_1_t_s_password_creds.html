Detailed Description

CredSSP TSPasswordCreds structure.

The TSPasswordCreds structure contains the user's password credentials that are delegated to the server.

The ASN.1 definition for the TSPasswordCreds structure is defined in MS-CSSP 2.2.1.2.1 TSPasswordCreds_:: 

TSPasswordCreds ::= SEQUENCE {
        domainName  [0] OCTET STRING,
        userName    [1] OCTET STRING,
        password    [2] OCTET STRING
}
Parameters
domain_nameThe name of the user's account domain.
usernameThe user's account name.
passwordThe user's account password.
