Detailed Description

Success and error codes for X509 verification, as returned by the underlying X509_STORE_CTX_get_error() function and passed by pyOpenSSL to verification callback functions.

See OpenSSL Verification Errors <https://www.openssl.org/docs/manmaster/man3/X509_verify_cert_error_string.html#ERROR-CODES>_ for details.
