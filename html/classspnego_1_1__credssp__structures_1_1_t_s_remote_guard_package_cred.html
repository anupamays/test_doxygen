Detailed Description

CredSSP TSRemoteGuardPackageCred structure.

The TSRemoteGuardPackageCred structure contains credentials for a specific security package.

The ASN.1 definition for the TSRemoteGuardPackageCred structure is defined in MS-CSSP 2.2.1.2.3.1 TSRemoteGuardPackageCred_:: 

TSRemoteGuardPackageCred ::= SEQUENCE{
    packageName [0] OCTET STRING,
    credBuffer  [1] OCTET STRING,
}
Parameters
package_nameThe name of the packages for which these credentials are intended.
cred_bufferThe credentials in a format specified by the CredSSP server.
