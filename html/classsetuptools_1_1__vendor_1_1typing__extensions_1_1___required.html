Detailed Description

A special typing construct to mark a key of a total=False TypedDict as required.

For example: 

class Movie(TypedDict, total=False):

title Required[str] year int 

    m = Movie(
title='The Matrix', # typechecker error if key is omitted
year=1999,

)

There is no runtime checking that a required key is actually provided when instantiating a related TypedDict.
