NoReturn OpenSSL._util.exception_from_error_queue ( Type[Exception] exception_type)

Convert an OpenSSL library failure into a Python exception.

When a call to the native OpenSSL library fails, this is usually signalled by the return value, and an error code is stored in an error queue associated with the current thread. The err library provides functions to obtain these error codes and textual error messages.
