◆ python_2_unicode_compatible()

pip._vendor.urllib3.packages.six.python_2_unicode_compatible ( klass)

A class decorator that defines unicode and str methods under Python 2.

Under Python 3 it does nothing.

To support Python 2 and 3 with a single code base, define a str method returning text and apply this decorator to the class.
