◆ connection_from_url()

HTTPConnectionPool urllib3.connectionpool.connection_from_url ( str url,
**typing.Any kw )

Given a url, return an :class:.ConnectionPool instance of its host.

This is a shortcut for not having to parse out the scheme, host, and port of the url before creating an :class:.ConnectionPool instance.

Parameters
urlAbsolute URL string that must include the scheme. Port is optional.

\**kw Passes additional parameters to the constructor of the appropriate :class:.ConnectionPool. Useful for specifying things like

Parameters
timeout
maxsize
headers
etc.

Example::

>>> conn = connection_from_url('http://google.com/')
>>> r = conn.request('GET', '/')
