Detailed Description

CredSSP NegoData structure.

The NegoData structure contains the SPNEGO tokens, the Kerberos messages, or the NTLM messages. While the structure is a SEQUENCE OF SEQUENCE this class just represents an individual SEQUENCE entry.

The ASN.1 definition for the NegoData structure is defined in MS-CSSP 2.2.1.1 NegoData_:: 

NegoData ::= SEQUENCE OF SEQUENCE {
        negoToken [0] OCTET STRING
}
Parameters
nego_tokenOne or more SPNEGO tokens and all Kerberos or NTLM messages, as negotiated by SPNEGO.
