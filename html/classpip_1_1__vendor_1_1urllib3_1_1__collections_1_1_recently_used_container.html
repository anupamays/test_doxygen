Detailed Description

Provides a thread-safe dict-like container which maintains up to maxsize keys while throwing away the least-recently-used keys beyond maxsize.

Parameters
maxsizeMaximum number of recent elements to retain.
dispose_funcEvery time an item is evicted from the container, dispose_func(value) is called. Callback which will get called
