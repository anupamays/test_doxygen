bytes spnego._ntlm_raw.md4.md4 ( bytes data)

Python implementation of md4 hashing.

This is a pure Python implementation of the MD4 Hashing Algorithm_. Recent distributions of Linux ship with OpenSSL 3.x which has disabled the md4 engine so for backwards compatibility this must be manually implemented here to ensure NTLM continues to work.

.. _MD4 Hasing Algorithm: https://datatracker.ietf.org/doc/html/rfc1320
