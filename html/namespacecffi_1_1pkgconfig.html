◆ flags_from_pkgconfig()

cffi.pkgconfig.flags_from_pkgconfig ( libs)

Return compiler line flags for FFI.set_source based on pkg-config output.

Usage ...

ffibuilder.set_source("_foo", pkgconfig = ["libfoo", "libbar >= 1.8.3"])

If pkg-config is installed on build machine, then arguments include_dirs, library_dirs, libraries, define_macros, extra_compile_args and extra_link_args are extended with an output of pkg-config for libfoo and libbar.

Raises PkgConfigError in case the pkg-config call fails.
